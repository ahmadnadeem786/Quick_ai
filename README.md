## Installation

Clone the repository:

```bash
git clone https://github.com/ahmadnadeem786/Quick_ai.git
```

Navigate to the frontend directory:

```bash
cd front
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Deployment
